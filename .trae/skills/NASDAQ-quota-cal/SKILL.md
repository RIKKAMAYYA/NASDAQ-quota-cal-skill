# NASQ Skill - 纳斯达克100指数基金数据监控

## 功能概述

本 Skill 用于每日定时采集并推送国内跟踪纳斯达克100指数的基金数据，包括：

### 场外基金（天天基金网数据源）
- 基金名称
- 基金代码
- 每日申购限额
- 管理费率 / 托管费率
- 更新日期

### 场内 ETF（东方财富/同花顺数据源）
- ETF 名称
- ETF 代码（如 159941, 513100 等）
- 实时溢价率
- 成交额
- 最新价格

## 触发条件

- 用户询问"今天纳指基金限额"、"纳斯达克ETF溢价率"、"纳指100基金数据"
- 每天早上 9:00 自动推送
- 用户手动调用查看最新数据

## 使用方法

```bash
# 安装依赖
pip install -r requirements.txt

# 配置 Server 酱 Token
export SERVERCHAN_KEY="your_serverchan_key"

# 手动运行
python src/main.py
```

## 数据源说明

| 数据类型 | 数据源 | 采集方式 |
|---------|--------|---------|
| 场外基金 | 天天基金网 (fund.eastmoney.com) | HTTP API 抓取 |
| 场内 ETF | 东方财富 (quote.eastmoney.com) | HTTP API 抓取 |
| 补充数据 | 同花顺 | HTTP API 抓取 |

## 输出格式

推送内容包含：
1. 场外基金限额汇总表格
2. 场内 ETF 溢价率汇总表格
3. 高风险溢价 ETF 预警提醒
